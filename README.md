# 🏠 House Price Prediction

An end-to-end **Machine Learning web application** built using **Python, Scikit-learn, and Streamlit** that predicts house prices based on various property features.

The application allows users to estimate house prices, compare multiple machine learning models, visualize data insights, perform batch predictions using CSV files, and explore feature importance through an interactive dashboard.

# 📖 Usage

1. Enter the house details.
2. Click **Predict House Price**.
3. View the predicted house price.
4. Explore the Dashboard for model metrics.
5. Compare multiple regression models.
6. Upload a CSV file for batch prediction.
7. Download prediction results.
